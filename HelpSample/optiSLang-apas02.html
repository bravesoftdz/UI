Monitoring database file (omdb)

This file contains some or all of the following simulation results.

Data typeDescription
Parameter ManagerParameters that are used for the simulation
Design ContainerResult designs of algorithm or MOP
MetamodelsMetamodels generated by MOP
Approximated designsWith surrogate metamodels approximated designs by postprocessing
Validated designsValidated approximation designs (created by MOP)
Reference designDesign with reference values
Nominal designNominal design of robustness or reliability algorithms
Algorithm infoInformation about used algorithm or MOP
Postprocessing meta dataPostprocessing related user meta data like previously selected or deactivated designs, selected design set and selected dimensions.
Images and processesReferences of images and processes which have been registered at the postprocessing
